Tips for learning to program from scratch

I remember when I started to programming it was very difficult because I don't understand how to do it. Now after practicing and practicing, I understand more and can make my own websites.

My first tip is to start now, because start now, practice now, If you leave it for later it is wasted time, you need to practice now, programming now.

My second tip is you start with PHP because your learning curve is easy.

My third tip is if you don't understand something, read again, watch a video on youtube about it, google it but don't stop until you get it.
